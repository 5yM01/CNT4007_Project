BitTorrent

1. Start Peers
    a. Run StartRemotePeers
    b. Peer Initialize
        i. Peer reads Common.cfg: Sets Variables
        ii. Peer reads PeerInfo.cfg
            -Set Bitfield: All bits to 1 if '1', 0 if '0'
        iii. TCP Connections

2. P2P Exchange
    a. Handshake & Bitfield Exchange
    b. Neighbor Selection
        i. Choke/Unchoke
            -Calculate Download Rate
        ii. Optimistically Unchoked
            -Random among interested neighbor Peers
    c. Piece Exchange
        i. Interested & Not Interested
        ii. Request & Piece

3. Peer Termination
    a. Termination
        i. All Peers have downloaded the complete file
